# Quantium starter repo
This repo contains everything you need to get started on the program! Good luck!

## Dash sales visualiser

This project includes a Dash app that visualises daily Pink Morsel sales and marks the
price increase date (2021-01-15).

### Run locally

1. Activate your virtual environment.
2. Install dependencies:

	```bash
	pip install -r requirements.txt
	```

3. Start the app:

	```bash
	python app.py
	```

4. Open the URL shown in the terminal (usually http://127.0.0.1:8050).
